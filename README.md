## What This Is

This is the **interactive companion** to the book. While the [main repository](https://github.com/aaron-recompile/mastering-taproot) contains the manuscripts and reference code, this repo provides Jupyter notebooks where you can:

- **Run** real Bitcoin Script examples step by step
- **Modify** parameters (private keys, scripts, tree structures) and see results instantly
- **Experiment** with "what if" scenarios (What if I swap two scripts in the Merkle tree?)
- **Visualize** Taproot data structures interactively
